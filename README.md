# ⏰ Analog Clock

A simple and elegant **Analog Clock** built using HTML, CSS, and JavaScript.

## 📌 Features

- Real-time updates
- Minimalistic and responsive design

## 🛠️ Technologies Used

- **HTML** – Structure
- **CSS** – Styling
- **JavaScript** – Time functionality

## 🎯 How to Run Locally

Open `index.html` in your browser.
